Innings Insights for IPL

Select an option below to get started:

Score Prediction for 1st Innings:

Predict the score of the batting team in the 1st innings by providing the following details:

  • Batting Team: Choose the team currently batting.
  • Bowling Team: Select the team currently bowling.
  • Current Score: Enter the runs scored so far.
  • Overs Completed: Specify the number of overs that have been bowled.
  • City: Indicate the city where the match is being played.
  • Wickets Out: Number of wickets that have fallen.
  • Runs in Last 5 Overs: Provide the total runs scored in the last five overs.

Once you fill in these details, you can click the button below to predict the score.

Score Prediction

Win Prediction for 2nd Innings:

Estimate the probability of winning a match by a team in the 2nd innings by providing the following details:

  • Batting Team: Select the team currently batting.
  • Bowling Team: Select the team currently bowling.
  • City: Indicate the city where the match is being played.
  • Overs Completed: Enter the number of overs that have been bowled.
  • Wickets Out: Number of wickets that have fallen.
  • Target Score: Specify the target score set by the opposing team.

After entering these details, click the button below to get the win prediction.
